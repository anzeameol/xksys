# Introduction
实现了一个选课系统网站

- Web容器: Tomcat
- 编程语言: Java
- 数据库连接池: DBCP
- 数据库操作: DBUtils
- 动态网页: JSP/Servlet
- 数据库: MySQL
- 前端美化: CSS
- 表单验证: jquery-validation

主要功能是登录，修改个人信息，学生选课/退课/查看分数，老师给分，管理员管理课程

此外还实现了MD5密码加密，查询功能，不符合规范的数据的提示，分页功能（逻辑分页）
